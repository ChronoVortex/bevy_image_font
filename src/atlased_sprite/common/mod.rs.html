1//! Shared utilities and constants for example binaries.
2//!
3//! This module provides common functionality and configuration shared across
4//! the example binaries for the project. It includes reusable constants,
5//! such as default text and color palettes, that are utilized to ensure
6//! consistency and reduce duplication across examples.
7//!
8//! # Key Features
9//! - **Default Text:** Includes a pangram for rendering demonstrations and
10//!   testing.
11//! - **Font Configuration:** Provides the font width of the example font.
12//! - **Rainbow Colors:** Supplies a palette of colors for visual styling in
13//!   examples.
14//!
15//! # Usage
16//! This module is intended for internal use by example binaries. It reduces
17//! redundancy by centralizing common assets and configurations. Depending on
18//! the active feature set, some utilities may not be used in certain examples.
19
20#![allow(
21    dead_code,
22    reason = "private utility code that, depending on the activated feature set, \
23    will sometimes be missing uses"
24)]
25
